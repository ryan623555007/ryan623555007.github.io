Urban Observatory Metadata Knowledge Graph
working with Polo Missier
at Newcastle University

The University of Newcastle has the largest Observatory in UK and astronomical data is open to the public. 357 sensors are installed in the city. However, the credibility of current astronomical data is not high enough, leading some companies to abandon the use of these data. In this project, we try to use the knowledge graph to describe the relationship between metadata and original data to improve the credibility of query result.

We establish a query system to support a closed loop of the whole process, from search to graph. Users can query the knowledge graph and a data description in JSON format. I use regular expressions to clean up astronomical data. Then combined the relationship between data and metadata. Finally, encapsulating the data interface becomes an API open to the public. I try to use the query results to improve users' trust in astronomical data, so that more people can use these data in industrial production. Some companies use the astronomical data to predict future weather. A total of 1.4 billion data were downloaded 7 million times by 2020.

Anti harassment AI competition
at Chinatelecom

Covid-19 broke out all over the world in 2020. So many companies have to increase telephone sales to maintain revenue. However, this cause more harassing calls. I use machine learning algorithms to predict harassment calls and push reminders to users' mobile phones.

Training set data is 100 million multi classification data, including user coordinates, personal information. In the preliminaries, I tried to use random forest algorithm to classify. Because of the inconsistent distribution of test set and training set, it did not perform well on prediction set. Then I try to use LGB model to adjust the weight of multi classification, and add e-commerce data to improve the quality of features, and the model has achieved good results. In the final, I used the fake tag technique, after several iterations, I finally ranked 30th out of 800.
